# Notes App (Core Data + SwiftUI)

A simple Notes app built using SwiftUI and Core Data. This app allows users to create, edit, delete, and search notes with local persistence. It also supports marking notes as favorites for easy access.

---

## Features

- Add new notes  
- Edit existing notes  
- Delete notes  
- Search notes by title or content  
- Mark notes as favorites  
- Persistent storage using Core Data  
- Simple, minimal user interface  
- Light and Dark Mode support  

---

## Technologies Used

- Swift 5  
- SwiftUI  
- Core Data  
- MVVM Architecture  
- Xcode (macOS)  
- iOS 17+ / macOS 14+ Compatible
